Project Template

Features:

  • Node.js and Express.js
  • Database is MongoDB
  • One entity for end users, a collection of automobiles
  • Another entity to hold user accounts
  • Security is provided by Passport.js and JWT handling

Open the README.md document for how-to info.


Paths:

  • Post activate user account (/api/useraccounts/activate)
  • Post register user account (/api/useraccounts/register)
  • Post login (/api/useraccounts/login)
  • Get info about me; it will return the token contents (/api/useraccounts/me)
  • Get all user accounts (/api/useraccounts)
  • Get one user account for dev testing only; disable or protect when deployed(/api/useraccounts/:id)
  • Get all people (/api/persons)
  • Get all people with auth (/api/personsprotected)
  • Get one person (/api/persons/:id)
  • Post add new person (/api/persons)
  • Put edit one person (/api/persons/:id)
  • Delete one person (/api/persons/:id)
  • Get all feed items (/api/feed)
  • Get all active feed items (/api/feed/active)
  • Get all news (/api/feed/news)
  • Get all events (/api/feed/events)
  • Get all notices (/api/feed/notices)
  • Get all alerts (/api/feed/alerts)
  • Get one feed item (/api/feed/:id)
  • Get all active feed items (/api/feed/active)
  • Post add new feed item (/api/feed)
  • Put edit one feed item (/api/feed/:id)
  • Delete one feed item (/api/feed/:id)
  • Get all text content (/api/textcontent)
  • Get one text content (/api/textcontent/:id)
  • Post add new text content (/api/textcontent)
  • Put edit one text content (/api/textcontent/:id)
  • Delete one text content(/api/textcontent/:id)
